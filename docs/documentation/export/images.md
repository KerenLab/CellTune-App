## Export Images
&nbsp;

Image export is under active development. 

You can currently export the viewer's selected image to a `.png`. 

![Menu_Export_Image](/assets/documentation/Menu_Export_Image.png){: width="35%"}  

The folder `/Images/Exports` is conveniently created for you to organize your exported images. 

![Export_Image_Files](/assets/documentation/Export_Image_Files.png){: width="65%"}
