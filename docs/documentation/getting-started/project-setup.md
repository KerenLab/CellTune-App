## Project Setup

### Starting a New Project
&nbsp;  

open application
startup screen
will see buttons for new project and load project
[can select a project from Recent projects list and click load project]
Select new project to open the new project setup dialog

### B. Set project directory
click on browse to set the project directory
you need to create a new folder
recommended location is locally 
not dropbox sync (if so, should turn sync off until the end of new project setup)


### Continuing a Pre-Existing Project

- can load project
- add a new batch of images
- replace images with newly cleaned images
- replace segmentation
