# Welcome
Welcome to the CellTune tutorial! This guide will walk you through the essential steps to begin analyzing spatial proteomics data with CellTune. By the end of this tutorial, you'll be familiar with the core workflow: loading data, exploring your images, classifying cells, and exporting results.

This tutorial uses demo data from a GVHD study. Follow the steps in order to get the most out of your learning experience.

If you haven't already installed the CellTune software, please start with the [Download & Install page](/download) before continuing.
